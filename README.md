# PDF-Toolkit

A lightweight Python tool to manage PDF files efficiently.

## Features
* Merge: Combine multiple PDFs into one.
* Split: Separate every page into its own file.
* Extract: Pull specific page ranges (e.g., pages 3-5).
* Encrypt: Password protect your documents.

## How to Download the Application
Go to the [Releases Page]to download the Windows App (.exe).

## How to Run (Source Code)
To run the raw script:
1. Install pypdf library. Run 'pip install pypdf' in Command Prompt 
2. Then run the script: 'python "PDF TOOLKIT.py"'
